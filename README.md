Credit Card Fraud Detection
📌 Overview

This project focuses on detecting fraudulent credit card transactions using Machine Learning techniques. The goal is to build a robust classification model that can accurately distinguish between legitimate and fraudulent transactions, helping financial institutions reduce losses and improve security.

🎯 Problem Statement

Credit card fraud is a significant challenge due to:

Highly imbalanced data (fraud cases are rare)
Real-time detection requirements
Need for high precision to minimize false positives

This project addresses these challenges using data preprocessing, feature engineering, and machine learning models.

🛠️ Tech Stack
Programming Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
Models : Random Forest,SMOTE
Tools: Visual studio Code 

⚙️ Project Workflow
Data Collection
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Handling Imbalanced Data
Model Training (Random Forest)
Model Evaluation (Accuracy, Precision, Recall, F1-score)
